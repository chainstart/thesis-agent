# Iteration 002: 首版格式修复器

日期：2026-05-06

## 新增能力

- 新增 `fix-format` 命令，生成修复稿，不覆盖原始论文。
- 新增 OOXML 修复模块 `format_fix.py`：
  - 删除文档尾部空白段落和空白分节段落。
  - 将连续空段压缩到 1 个以内，避免靠空段控制版式。
  - 将正文第一节页码从 `1 绪论` 开始重置为 1。
  - 将主章节标题套用文档内已有 `heading 1` 样式。
  - 对图表题注添加基本 keep 规则，降低题注与对象分离概率。
- 审计报告新增页面标号识别，能检查“物理页”和“页码标签”是否一致。
- 优化近空白页判断，避免把目录尾页误报成空白页。
- 如果存在疑似题注孤页，审计状态保持 `needs_fix`。

## 杨钰婷样本修复验证

命令：

```bash
PYTHONPATH=src python3 -m thesis_agent fix-format \
  --target samples/drafts/物联网2212-杨钰婷-毕业论文初稿.docx \
  --output runs/fixes/yang-fixed.docx \
  --template samples/templates/论文格式.doc \
  --audit-out runs/fixes/yang-fixed-audit
```

修复动作：

- 删除尾部空段：3 个。
- 删除尾部空白分节段落：2 个。
- 删除长空段中的多余空段：1 个。
- 正文页码重置：已应用。
- 主章节标题样式：3 处。
- 题注 keep 规则：18 处。

渲染审计结果：

- 页数：47 页 -> 45 页。
- 空白页：46、47 -> 无。
- 近空白页：4、46、47 -> 无。
- `1 绪论`：物理第 5 页，页码标签 1。
- 长空段：存在 1 组 -> 无。
- 剩余问题：疑似题注孤页仍需进一步修复，测试章节内容偏薄。
